Generate Attendance

After long lectures, the last thing anyone wants is to sit around while attendance lists are manually typed and formatted. Attendance Buddy is a tiny automation tool that simplifies this everyday task.

Just select the date, course name, course code, and professor, and the tool instantly generates a clean, ready-to-send attendance list.

Simple. Fast. Done in seconds.

Why I Built This

Every class ends the same way — roll numbers collected, details written down, and someone manually formatting the attendance list before sending it to the professor.

It’s repetitive, boring, and honestly unnecessary.

So instead of doing it the long way every day, I built a small tool that does the formatting instantly.

A tiny automation, but it saves time every single lecture.
